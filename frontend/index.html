What it does

One agent for all your resources

🔍

Smart Source Detection

Two-stage detection: regex rules first, LLM classifier fallback for ambiguous inputs.

⚡

Fully Async Pipeline

detect → extract → clean → enrich → summarize → store. No blocking I/O anywhere.

🗄

Persistent Vault

Every resource saved with title, snippet, source type, raw data, and LLM summary.

🧩

Modular Processors

Dedicated processor per source. Easy to extend with new integrations.

🖼

Multi-Input Support

Mix URLs, text notes, and image uploads in a single message. Gets a combined insight.

🤖

Local LLM Ready

Runs fully offline with Ollama (qwen2.5, mistral). No API keys needed locally.

Get started locally

Install in 5 steps

1

Clone the repository

git clone https://github.com/adityaksx/ai_resource_agent.git && cd ai_resource_agent
2

Install Python dependencies

pip install -r requirements.txt

For full OCR + AI: also run pip install pillow pytesseract paddleocr openai-whisper torch

3

Install system dependencies

sudo apt install tesseract-ocr ffmpeg # Linux / WSL
4

Start Ollama (local LLM)

ollama serve ollama pull qwen2.5:7b-instruct
5

Run the web interface

uvicorn web.app:app --reload

Then open http://localhost:8000

Browse resources already saved in the demo vault, or start a new chat.
